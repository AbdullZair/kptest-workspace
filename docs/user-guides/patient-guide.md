# Patient Guide - KPTEST System

## Welcome to KPTEST

KPTEST is a telemedicine platform that helps you manage your therapy program from anywhere. This guide will help you get started and make the most of the system.

## Getting Started

### Downloading the Mobile App

#### iOS (iPhone/iPad)
1. Open the App Store
2. Search for "KPTEST"
3. Tap "Get" to download
4. Requires iOS 14.0 or later

#### Android
1. Open Google Play Store
2. Search for "KPTEST"
3. Tap "Install"
4. Requires Android 8.0 or later

### Creating Your Account

**Method 1: Registration Link from Coordinator**

1. Open the registration link sent to your email
2. Enter your details:
   - PESEL (Polish national ID)
   - Email address
   - Phone number
   - Create password (min 8 characters, include uppercase, lowercase, and number)
3. Verify your email by clicking the link sent
4. Complete your profile

**Method 2: Self-Registration**

1. Open the KPTEST app
2. Tap "Register"
3. Enter your details
4. Wait for coordinator approval

### First Login

1. Open the KPTEST app
2. Enter your email and password
3. If 2FA is enabled, enter the code from your authenticator app
4. You're now logged in!

### Completing Tasks

1. Tap on "Tasks" from the bottom menu
2. Find today's tasks
3. Tap on a task to view details
4. Complete the task as instructed
5. Tap "Mark as Complete"

**Task Types:**
- 📝 **Exercise** - Complete therapy exercises
- 📖 **Reading** - Read educational materials
- 🎧 **Listening** - Listen to audio exercises
- 📹 **Video** - Watch therapy videos
- 📊 **Assessment** - Complete questionnaires

### Viewing Educational Materials

1. Tap on "Materials" from the bottom menu
2. Browse available materials
3. Tap on a material to view
4. Download for offline viewing (optional)

**Material Types:**
- PDF documents
- Videos
- Audio files
- Interactive exercises

### Messaging Your Therapy Team

1. Tap on "Messages" from the bottom menu
2. Select a conversation or start a new one
3. Type your message
4. Attach files if needed (max 10MB)
5. Tap "Send"

**Response Time:**
- Urgent matters: Call your coordinator
- General questions: Expect response within 24-48 hours

### Calendar and Events

1. Tap on "Calendar" from the bottom menu
2. View upcoming appointments
3. Tap on an event for details
4. Set reminders for important events

**Event Types:**
- Individual therapy sessions
- Group sessions
- Assessment appointments
- Important deadlines

## Profile Settings

### Updating Your Information

1. Tap on "Profile" from the bottom menu
2. Tap "Edit Profile"
3. Update your information
4. Tap "Save"

**You can update:**
- Phone number
- Email address
- Emergency contact
- Profile photo

### Notification Settings

1. Go to Profile → Settings → Notifications
2. Choose which notifications to receive:
   - Task reminders
   - New messages
   - Appointment reminders
   - Project updates

### Security Settings

#### Changing Password

1. Go to Profile → Settings → Security
2. Tap "Change Password"
3. Enter current password
4. Enter new password (twice)
5. Tap "Save"

#### Setting Up Two-Factor Authentication (2FA)

1. Go to Profile → Settings → Security
2. Tap "Enable 2FA"
3. Scan the QR code with your authenticator app:
   - Google Authenticator
   - Authy
   - Microsoft Authenticator
4. Enter the verification code
5. Save your backup codes in a safe place

#### Biometric Login

1. Go to Profile → Settings → Security
2. Tap "Enable Biometric Login"
3. Authenticate with Face ID or Touch ID
4. You can now use biometrics to log in

## Tips for Success

### Building Good Habits

1. **Set a routine** - Complete tasks at the same time each day
2. **Use reminders** - Enable push notifications
3. **Start small** - Focus on completing daily tasks
4. **Track progress** - Check your adherence score regularly
5. **Ask for help** - Message your therapy team if struggling

### Understanding Adherence Score

Your adherence score shows how consistently you complete tasks:

| Score | Status | Meaning |
|-------|--------|---------|
| 90-100% | Excellent | Outstanding commitment |
| 75-89% | Good | On track with therapy |
| 60-74% | Fair | Room for improvement |
| Below 60% | Needs Attention | Contact your coordinator |

**How it's calculated:**
```
Adherence = (Completed Tasks / Assigned Tasks) × 100%
```

### Offline Mode

The app works offline:
- View downloaded materials
- Complete tasks (syncs when online)
- View cached information

**To use offline:**
1. Download materials when online
2. Complete tasks as usual
3. Data syncs automatically when you're back online

## Troubleshooting

### Common Issues

**Can't log in:**
- Check your internet connection
- Verify email and password
- Try "Forgot Password" if needed
- Contact support if issue persists

**App crashes:**
- Close and reopen the app
- Check for app updates
- Restart your device
- Reinstall if problem continues

**Tasks not syncing:**
- Check internet connection
- Pull down to refresh
- Wait a few minutes and try again

**Can't download materials:**
- Check storage space on device
- Ensure you have internet connection
- Try again later
