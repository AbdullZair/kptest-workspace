# Doctor/Therapist Guide - KPTEST System

## Overview

As a Doctor/Therapist in KPTEST, you have access to patient information, can communicate with patients, and track therapy progress for patients assigned to your projects. This guide covers all therapist functions.

## Getting Started

### First Login

1. Navigate to https://app.kptest.com
2. Enter your credentials provided by administrator
3. Complete 2FA setup if prompted
4. Review your assigned projects on dashboard

## Patient Management

### Viewing Assigned Patients

1. Navigate to Patients
2. View list of patients assigned to your projects
3. Use filters to find specific patients:
   - By project
   - By name
   - By adherence level
   - By status

### Patient Profile

Click on a patient to view:

**Personal Information:**
- Name, age, contact
- Emergency contact
- Medical record number

**Therapy Information:**
- Assigned projects
- Current tasks
- Adherence history
- Progress charts

**Communication:**
- Message history
- Recent interactions
- Notes from team

### Patient Progress Tracking

**Adherence Overview:**

1. Open patient profile
2. Go to "Progress" tab
3. View adherence chart:
   - Daily completion rate
   - Weekly trends
   - Comparison to project average

**Task History:**

1. Open patient profile
2. Go to "Tasks" tab
3. View:
   - Completed tasks
   - Pending tasks
   - Overdue tasks
   - Task details

## Project Management

### Viewing Project Details

1. Navigate to Projects
2. Click on your project
3. View project overview:
   - Patient list
   - Overall adherence
   - Task templates
   - Team members
   - Calendar

### Project Reports

**Adherence Report:**

1. Open project
2. Go to "Reports" tab
3. Select "Adherence Report"
4. View:
   - Patient adherence list
   - Average adherence
   - Patients below threshold
   - Trends over time

**Progress Report:**

1. Open project
2. Go to "Reports" tab
3. Select "Progress Report"
4. View:
   - Task completion rates
   - Milestone achievements
   - Patient engagement metrics

### Adding Clinical Notes

1. Open patient profile
2. Go to "Notes" tab
3. Click "Add Note"
4. Enter note:
   - Date
   - Session type
   - Observations
   - Recommendations
5. Click "Save"

**Note Types:**
- Session notes
- Progress notes
- Assessment notes
- Intervention notes

## Communication

### Messaging Patients

1. Navigate to Messages
2. Click "Compose"
3. Select patient recipient
4. Enter subject and message
5. Click "Send"

**Message Guidelines:**
- Use professional language
- Response within 24 hours
- Document clinical advice in notes
- Escalate urgent matters

### Messaging Team

1. Navigate to Messages
2. Click "Compose"
3. Select team members
4. Discuss patient care
5. Coordinate therapy

### Announcements

For project-wide messages:

1. Navigate to project
2. Go to "Communication" tab
3. Click "Announcement"
4. Compose message
5. Send to all project members

## Calendar and Scheduling

### Viewing Schedule

1. Navigate to Calendar
2. View your appointments
3. Filter by:
   - Individual sessions
   - Group sessions
   - Assessments
   - Availability

### Creating Appointments

1. Navigate to Calendar
2. Click "Add Event"
3. Fill in details:
   - Patient(s)
   - Date and time
   - Type
   - Location
   - Notes
4. Click "Create"
5. Patient receives notification

### Managing Availability

1. Go to Profile → Availability
2. Set regular hours
3. Mark time off
4. Block unavailable times

## Assessment and Evaluation

### Creating Assessments

1. Open project → Assessments tab
2. Click "Create Assessment"
3. Define assessment:
   - Title
   - Questions
   - Scoring criteria
   - Due date
4. Assign to patients

### Reviewing Assessment Results

1. Open project → Assessments
2. Select completed assessment
3. View results:
   - Individual scores
   - Average scores
   - Response distribution
4. Export if needed

### Progress Evaluation

**Individual Patient:**

1. Open patient profile
2. Go to "Evaluations" tab
3. Review:
   - Baseline assessment
   - Progress assessments
   - Current status
4. Document evaluation notes

**Project-wide:**

1. Open project → Reports
2. Select "Evaluation Summary"
3. View aggregate progress
4. Identify patterns

3. Select "Progress Report"
4. Choose date range
5. Generate and review

When referring to specialist:

1. Open patient profile
2. Go to "Referrals" tab
3. Click "Create Referral"
4. Include:
   - Reason for referral
   - Relevant history
   - Current therapy
   - Specific questions
5. Send to specialist

## Best Practices

### Patient Communication

1. **Professional Tone** - Maintain appropriate boundaries
2. **Clear Instructions** - Explain therapy tasks clearly
3. **Timely Responses** - Reply within 24 hours
4. **Encouraging** - Provide positive reinforcement
5. **Document** - Record all clinical communications

### Clinical Documentation

1. **Timely** - Document sessions promptly
2. **Accurate** - Record objective observations
3. **Complete** - Include all relevant information
4. **Confidential** - Maintain privacy standards
5. **Compliant** - Follow regulatory requirements

### Therapy Management

1. **Monitor Adherence** - Review patient engagement regularly
2. **Adjust as Needed** - Modify therapy based on progress
3. **Communicate with Team** - Coordinate with coordinators
4. **Follow Protocols** - Adhere to clinical guidelines
5. **Continuing Education** - Stay updated on best practices

## Privacy and Compliance

### Data Access

- Access only assigned patients
- Use secure connections
- Log out when finished
- Report suspicious activity

### Documentation Standards

- Use professional language
- Avoid abbreviations
- Date and sign all entries
- Never alter past entries (add addendum)

### Mandatory Reporting

Report immediately if:
- Patient safety concern
- Suspected abuse/neglect
- Adverse events
- Data breach

## Troubleshooting

### Common Issues

**Can't see patient:**
- Verify patient is assigned to your project
- Check patient status is active
- Contact coordinator if needed

**Can't access report:**
- Verify you have project access
- Check report date range
- Try refreshing page

**Message not sending:**
- Check internet connection
- Verify recipient is active
- Try again or contact support
